# DevToolkit 🛠️

Your all-in-one digital workspace with AI assistance. A comprehensive toolkit for developers, creators, and productivity enthusiasts.

## Features

### 💻 Code Editor
- Syntax highlighting for HTML, CSS, JS
- Live preview
- Code templates
- AI code generation
- Bracket matching
- Multiple themes

### 📝 Notes
- Rich text editing
- Markdown support
- Tags and search
- AI writing assistant
- Export to Markdown/HTML

### 📋 Projects
- Status workflow (Planning → In Progress → Review → Done)
- Tags and deadlines
- Progress tracking

### ✅ Tasks
- Priority levels (High/Medium/Low)
- Due dates
- Pomodoro timer
- AI task suggestions
- Filter and organize

### 🎨 Design
- Figma-style design canvas
- Shape tools (rectangle, ellipse, line, text)
- Layer management
- Export to PNG/SVG/CSS
- Color picker

### 🎬 YouTube
- Script planner
- Thumbnail ideas
- Upload scheduler
- Content calendar
- SEO checklist

### 📺 Streaming
- Scene switcher
- Chat commands
- Alert simulator
- Timer/countdown
- Widgets

### 🤖 AI Assistant
- Powered by Claude 3 Haiku (Cloud) or Ollama (Local)
- Context-aware conversation
- Works offline with local fallback
- Multi-model support

## Themes

- 🌙 Dark (default)

## Getting Started

1. Open `index.html` in your browser
2. For AI features, get a free API key from [OpenRouter](https://openrouter.ai/keys)
3. Start creating!

## Local Development

```bash
# Using Python
python -m http.server 8080

# Using Node.js
npx serve
```

Then open `http://localhost:8080`

## Deploy to GitHub Pages

This project includes automatic GitHub Actions deployment.

1. Fork/clone this repo
2. Push to `main` branch
3. GitHub Actions will deploy automatically

## Tech Stack

- **Frontend**: Vanilla HTML/CSS/JS
- **AI**: OpenRouter API / Ollama
- **Storage**: localStorage
- **Icons**: Font Awesome
- **Fonts**: Inter, Space Grotesk, JetBrains Mono
